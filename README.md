# AFOP Banshee Skin Color Editor
Edit ikran (banshee) skin color pattern files (`.mcolorpattern`) for Avatar: Frontiers of Pandora.

## How to Use

1. **Load files** — each panel (**HEAD** / **BODY**) has its own **Load** button. Banshee patterns come in pairs, so load the matching head and body files side by side.

2. **Edit a color** — click any color swatch to open the color picker.
   - Drag the **gradient square** to set saturation/brightness.
   - Drag the **rainbow bar** to shift hue.
   - Type a **hex code** directly or adjust the **R G B** spinboxes.

3. **Save** — click **Save** on the panel you want to write. Each panel saves independently.

> Each pattern has 10 color slots (Color 1–10). Only the hex color values are changed — everything else in the file is left byte-for-byte identical.
